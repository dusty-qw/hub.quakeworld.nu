## Install
1. `git clone git@github.com:vikpe/qwsb.git`
1. `cd qwsb`   
1. `yarn install`

---

## Development
Launch dev server at `http://localhost:3000` with HMR-enabled.
```
yarn watch
```

---

## Production
Build app to `/build`
```
yarn build
```
